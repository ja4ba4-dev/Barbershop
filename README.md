# Barbershop-1C

---

### **Обзор проекта**

Моделирует бизнес-процессы барбершопа:  
учёт товаров и услуг по партиям и срокам годности,  
контроль остатков и себестоимости,  
расчёт выручки и прибыли,  
регламентированный учёт по РСБУ,  
начисление зарплаты (оклад, премии, больничные),  
мобильная запись клиентов с аутентификацией и передачей данных в основную базу через **REST**.

---

### **Техническая архитектура**

- **Платформа:** 1C:Предприятие 8.3.24+ (управляемые формы, СКД 2.0, мобильная платформа 8.3.22+)  
- **СУБД:** Файловая (локально); MSSQL 2019+ для серверного режима (скрипты в `/SQL/`)  
- **Язык:** Встроенный 1C (модули объектов и форм), язык запросов 1С, запросы к виртуальным таблицам  
- **Мобильное приложение:** 1C Мобильная платформа, эмулятор Android Studio, сборка APK
