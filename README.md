<div class="section">
  <div class="header">Barbershop-1C</div>
  <hr class="divider">
  <p><strong>Обзор проекта</strong>
    <br> <br>
Моделирует бизнес-процессы барбершопа: учёт товаров/услуг по партиям/срокам годности, контроль остатков/себестоимости, расчёт выручки/прибыли, регламентированный учёт по РСБУ, начисления зарплаты (оклад, премии, больничные), мобильная запись клиентов с аутентификацией и передачей данных в основную базу via REST.</p>
</div>

<div class="section">
  <div class="header">Техническая архитектура</div>
  <ul>
    <li><strong>Платформа:</strong> 1C:Предприятие 8.3.24+ (управляемые формы, СКД 2.0, мобильная платформа 8.3.22+)</li>
    <li><strong>СУБД:</strong> Файловая (локально); MSSQL 2019+ для серверного (скрипты в /SQL/)</li>
    <li><strong>Язык:</strong> Встроенный 1C (модули объектов/форм), язык запросов 1С, запросы к виртуальным таблицам</li>
    <li><strong>Мобильное приложение:</strong> 1C Мобильная платформа, эмулятор Android Studio, сборка APK</li>
  </ul>
</div>
